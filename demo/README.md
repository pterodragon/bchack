## Installation
```bash
npm run clean
npm boostrap
npm install
npm run build
```

## Run seeder server
```bash
cd webtorrentx-paid
# use the appropiate .env file
npm run stat-seeder
```

## Run leecher client ui
```bash
cd client-ui
# use the appropiate .env file
npm run start
```

then access [http://localhost:3000](http://localhost:3000)
