## Installation
```bash
npm run clean
npm install
npm boostrap
npm prepare
npm run build
```

## Run seeder server
```bash
cd packages/webtorrentx-paid
# use the appropiate .env file
npm run stat-seeder
```

## Run leecher client ui
```bash
cd packages/client-ui
# use the appropiate .env file
npm run start
```

then access [http://localhost:3000](http://localhost:3000)
